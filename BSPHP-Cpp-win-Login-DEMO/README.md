# BSPHP-Cpp-win-Login-DEMO

![目录结构图](../docs/project-structure.svg)

登录模式 Win32 GUI 演示，包含账号登录、短信/邮箱登录、注册、找回、反馈、登录后控制台等。

## 配置初始化文件

- 地址与密钥初始化文件：`core/login_demo_config.cpp`
- 主要配置项：
  - `kBSPHPURL`
  - `kMutualKey`
  - `kServerKey` / `kClientKey`
  - `kCodeURLPrefix`

## 说明

- 界面资源说明可参考：`配置说明/`
- 可执行入口：`main.cpp`
